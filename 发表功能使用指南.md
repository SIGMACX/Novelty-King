# 论文发表功能使用指南

## 🎉 新功能说明

管理员现在可以将审核通过的论文发表到网站上，公开展示给所有访问者。

---

## 📋 设置步骤

### 第一步：执行数据库更新

在 Supabase SQL Editor 中执行 `add-published-fields.sql` 文件的内容：

```sql
-- 添加发表相关的字段到 submissions 表

-- 添加发表日期字段
ALTER TABLE submissions
ADD COLUMN IF NOT EXISTS published_at TIMESTAMP WITH TIME ZONE;

-- 添加是否公开显示的字段
ALTER TABLE submissions
ADD COLUMN IF NOT EXISTS is_published BOOLEAN DEFAULT false;

-- 更新状态检查约束，添加 'published' 状态
ALTER TABLE submissions
DROP CONSTRAINT IF EXISTS submissions_status_check;

ALTER TABLE submissions
ADD CONSTRAINT submissions_status_check
CHECK (status IN ('pending', 'under_review', 'accepted', 'rejected', 'published'));

-- 创建索引以提高查询已发表文章的性能
CREATE INDEX IF NOT EXISTS idx_submissions_published ON submissions(is_published, published_at DESC);

-- 添加策略：公开访问已发表的文章
CREATE POLICY "Anyone can view published submissions" ON submissions
  FOR SELECT
  USING (is_published = true);
```

## 👨‍💼 管理员操作流程

### 1. 登录管理后台

1. 登录您的管理员账号
2. 点击右上角用户菜单 → **管理后台**
3. 访问 http://localhost:3001/admin

### 2. 审核论文

在管理后台，您可以：

**查看所有投稿**
- 点击任一投稿查看详情
- 查看论文标题、作者、摘要、关键词
- 下载并阅读 PDF 文件

**更改审核状态**
- 待审核 → 审核中 → 已接受/已拒绝
- 点击对应的状态按钮即可

### 3. 发表论文

**发表条件：**
论文应该已经过审核（建议状态为"已接受"）

**发表步骤：**
1. 在管理后台选择要发表的论文
2. 滚动到"发表管理"部分
3. 点击 **🚀 发表论文** 按钮
4. 确认发表操作

**发表后：**
- 论文状态自动变为"已发表"
- 论文将在网站的"已发表论文"页面公开展示
- 所有访问者都可以查看和下载

### 4. 取消发表（可选）

如果需要撤下已发表的论文：
1. 在管理后台选择已发表的论文
2. 在"发表管理"部分点击 **取消发表**
3. 确认操作

取消后，论文将不再在公开页面显示。

## 🌐 公开展示页面

### 访问已发表论文

所有用户（包括未登录用户）都可以访问：
- **URL**: http://localhost:3001/publications
- **导航**: 点击顶部导航栏的 **Publications / 已发表**

### 展示内容

每篇已发表的论文显示：
- ✅ 论文标题
- 👤 作者信息（第一作者、通讯作者）
- 📅 发表日期
- 📝 摘要（如果有）
- 🏷️ 关键词标签
- 📁 研究领域
- 📥 PDF 下载按钮

### 用户操作

访问者可以：
1. 浏览所有已发表论文列表
2. 点击论文卡片查看详情
3. 下载PDF文件阅读完整内容
4. 通过关键词了解研究方向

## 📊 完整工作流程

### 论文生命周期

```
用户提交论文
    ↓
待审核 (pending)
    ↓
审核中 (under_review)  ← 管理员审阅中
    ↓
已接受 (accepted) / 已拒绝 (rejected)
    ↓
已发表 (published)  ← 管理员点击"发表"
    ↓
公开展示  ← 在 /publications 页面显示
```

### 各角色权限

**普通用户：**
- ✅ 提交论文
- ✅ 查看自己的投稿状态
- ✅ 浏览所有已发表论文
- ✅ 下载已发表论文的 PDF

**管理员：**
- ✅ 查看所有投稿
- ✅ 更改投稿状态
- ✅ 发表/取消发表论文
- ✅ 添加审核意见
- ✅ 下载所有 PDF

**访客（未登录）：**
- ✅ 浏览已发表论文
- ✅ 下载已发表论文的 PDF
- ❌ 无法提交论文
- ❌ 无法查看未发表的投稿

## 🔒 安全说明

### 数据隐私

- **未发表的论文**：只有作者本人和管理员可以查看
- **已发表的论文**：公开，所有人都可以访问
- **作者邮箱**：在已发表论文详情页显示，用于联系

### 发表控制

- 只有管理员可以执行发表操作
- 发表前会要求二次确认
- 发表后可以随时取消

## 💡 最佳实践

### 管理员建议

1. **发表前检查**：
   - 确认论文质量符合标准
   - 检查 PDF 文件可以正常打开
   - 验证作者信息准确
   - 确保摘要和关键词完整

2. **发表时机**：
   - 建议在"已接受"状态后再发表
   - 可以批量审核后统一发表

3. **内容管理**：
   - 定期检查已发表论文质量
   - 及时撤下有问题的论文
   - 保持发表论文的学术水准

### 用户建议

1. **提交论文时**：
   - 确保摘要简洁清晰（100字以内）
   - 添加准确的关键词
   - 选择正确的研究领域
   - PDF 格式规范

2. **查看已发表论文时**：
   - 尊重版权和学术规范
   - 引用时注明出处

## ❓ 常见问题

### Q: 如何添加管理员账户？

A: 在 Supabase SQL Editor 中执行：
```sql
-- 先查找用户 ID
SELECT id, email FROM auth.users WHERE email = 'admin@example.com';

-- 添加为管理员
INSERT INTO admin_users (user_id) VALUES ('用户ID');
```

### Q: 发表后可以修改论文吗？

A: 已发表的论文信息会被展示，如需修改：
1. 先取消发表
2. 联系作者修改后重新提交
3. 审核通过后再次发表

### Q: PDF 文件大小有限制吗？

A: 是的，最大 10MB。这是为了保证数据库性能和页面加载速度。

### Q: 已发表论文会被搜索引擎收录吗？

A: 取决于您的网站 SEO 设置。建议添加适当的 meta 标签以提高可见性。
