# 管理员账号设置说明

## ⚠️ 重要提示

由于 Supabase 项目配置限制，管理员账号**不能直接通过登录页面注册**。

错误信息：`Signups not allowed for otp`

## ✅ 正确的管理员设置流程

### 方法一：先作为作者注册，再升级为管理员（推荐）

#### 第一步：注册作者账户

1. 访问 http://localhost:3000/login
2. 点击"注册"
3. 输入邮箱和密码（例如：admin@example.com）
4. 完成注册流程

#### 第二步：在 Supabase 中查找用户 ID

1. 访问 https://supabase.com/dashboard/project/esoswgywjkkijwyvhwdl
2. 点击左侧 **SQL Editor**
3. 点击 **New Query**
4. 执行以下查询：

```sql
SELECT id, email, created_at
FROM auth.users
WHERE email = 'admin@example.com';  -- 替换为您的邮箱
```

5. 复制返回的 `id`（UUID 格式）

#### 第三步：添加到管理员表

在 SQL Editor 中执行：

```sql
INSERT INTO admin_users (user_id)
VALUES ('您复制的用户ID');
```

**示例**：
```sql
INSERT INTO admin_users (user_id)
VALUES ('123e4567-e89b-12d3-a456-426614174000');
```

#### 第四步：测试管理员登录

1. 访问 http://localhost:3000/admin/login
2. 输入管理员邮箱
3. 点击"发送验证码"
4. 查收邮箱中的验证码
5. 输入验证码登录
6. 应该成功跳转到管理后台

---

### 方法二：直接在 Supabase 创建用户

#### 第一步：在 Supabase 创建用户

1. 访问 https://supabase.com/dashboard/project/esoswgywjkkijwyvhwdl
2. 点击 **Authentication** → **Users**
3. 点击 **Add user**
4. 选择 **Create new user**
5. 输入邮箱地址（例如：admin@example.com）
6. 输入密码（可选，如果使用 OTP 登录则不需要）
7. 勾选 **Auto Confirm User**（自动确认用户）
8. 点击 **Create user**

#### 第二步：复制用户 ID

在用户列表中找到刚创建的用户，复制其 UUID

#### 第三步：添加到管理员表

在 SQL Editor 中执行：

```sql
INSERT INTO admin_users (user_id)
VALUES ('复制的用户ID');
```

#### 第四步：测试登录

访问 http://localhost:3000/admin/login 进行登录测试

## 🔍 验证管理员设置

执行以下 SQL 查看所有管理员：

```sql
SELECT
  au.user_id,
  u.email,
  au.created_at as admin_since
FROM admin_users au
JOIN auth.users u ON au.user_id = u.id
ORDER BY au.created_at DESC;
```

## ❓ 常见问题

### Q: 为什么不能直接在管理员登录页面注册？

**A**: Supabase 项目设置中禁用了 OTP 注册功能（`Signups not allowed for otp`）。这实际上是一个**安全特性**，确保只有通过作者注册流程（带密码）注册的用户，或由管理员手动创建的用户才能使用系统。

### Q: 如何启用 OTP 注册？

**A**: 如果您想启用 OTP 注册，可以在 Supabase 控制台进行设置：

1. 访问 https://supabase.com/dashboard/project/esoswgywjkkijwyvhwdl
2. 点击 **Authentication** → **Settings**
3. 找到 **Email Auth** 部分
4. 确保 **Enable email signups** 已启用

但**不推荐**这样做，因为当前的限制实际上提供了更好的安全性。

### Q: 管理员登录时需要密码吗？

**A**: 不需要。管理员使用 **邮箱 OTP 验证码**登录，每次登录时会收到一个 6 位验证码。这比密码更安全，因为验证码每次都不同且有时效性。

### Q: 如果忘记是否已添加为管理员？

**A**: 在 SQL Editor 中执行：

```sql
SELECT * FROM admin_users WHERE user_id = '您的用户ID';
```

如果返回结果，说明已是管理员；如果没有结果，需要重新添加。
